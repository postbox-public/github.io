Keyboard Shortcuts

Note: If you're used to Gmail's keyboard shortcuts and would prefer to use them instead of Postbox's shortcuts, go to Postbox (macOS)/Tools (Windows) > Preferences/options > Advanced and checkmark Use Gmail shortcuts. For a list of Gmail's shortcuts, click here. (Many of the keyboard shortcuts in Postbox have no equivalent in Gmail, so they remain as shown below.) To restore the native Postbox shortcuts, uncheck the Use Gmail shortcuts box.
